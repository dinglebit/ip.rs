# my-ip

Get your public IP Address from the command line.

```sh
$ cargo install my-ip
$ ./my-ip
IP:            8.8.8.8
Latitude:      0.0
Longitude:     0.0
Country Code:  US
Country:       United States
City:          Anytown
Continent:     North America
Time Zone:     America/Denver
Postal Code:   8888
Organization:  ISP
ASN:           ASN134
Subdivision:   Colorado
```
